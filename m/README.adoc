= m => direct messages

* private messages using NIP-17 (gift wrap / seal / rumor)
* three-layer encryption: rumor (kind 14) wrapped in seal (kind 13) wrapped in gift wrap (kind 1059)
* timestamps are randomised 0-48h into the past for privacy (NIP-59)
* conversations are persisted locally and restored on load
* pending (undecrypted) gift wraps are persisted and restored across sessions

to subscribe to DMs from relays, type: `. m get`

* uses your kind 10050 relays if available, falls back to read relays
* kind 10050 relays authenticate automatically (NIP-42)
* set `m_get` option to `on` to auto-subscribe on load: `. o add m_get on`
* set `m_decrypt` option to `on` to auto-decrypt incoming gift wraps: `. o add m_decrypt on`

to start a new conversation, type: `. m new <npub|pubkey>`

* typing in the CLI while a conversation is open sends a DM to that person
* by default a draft is shown for review before sending (send / cancel)
* set `m_auto_send` option to `on` to skip the draft step: `. o add m_auto_send on`

to decrypt a specific gift wrap by id, type: `. m decrypt <id>`

* the inspect button (i) on each message shows the full decrypt chain (rumor, seal, wrap ids)
* the mark read button in the conversation header toggles visibility of older messages
* failed decrypts are logged and won't reappear on subsequent `get` calls
