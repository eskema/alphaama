= o => options

Manages localStorage options and tab-scoped session overrides.

== localStorage options (`o add`)

* `.o add key value` — set a localStorage option
* `.o add key` — set to empty (clears the value)
* `.o reset` — reset all options to defaults
* `.o reset key` — reset a single key to its default
* `.o del key` — delete a non-default option

== session overrides (`o ss`)

Session overrides are tab-scoped and take priority over localStorage values without modifying them.
They persist across page reloads within the same browser session.

* `.o ss key value` — override a localStorage key for this tab only
* `.o ss key` — override with empty value (disables, e.g. `on_load_sub`)
* Running `.o add key value` on a key with a session override clears the override automatically

The options panel shows session overrides above the localStorage list.
Overridden localStorage items appear dimmed with strikethrough.

== options

* `theme` (default `dark`) — `dark` or `light`
* `score` (default `11`) — minimum trust score to embed media
* `pagination` (default `100`) — number of root events shown
* `pow` (default `0`) — proof of work difficulty
* `reaction` (default `🤘`) — default reaction emoji
* `on_load_sub` (default empty) — sub prefix to run on load (empty = disabled)
* `auto_decrypt` (default `off`) — `on`, `on_view`, or `off`
* `m_decrypt` (default `off`) — auto-decrypt DMs
* `m_get` (default `off`) — auto-fetch DMs
* `m_auto_send` (default `off`) — auto-send DMs
* `cash` (default `on`) — enable service worker cache
* `ext_image` (default `gif heic jpeg jpg png webp`) — allowed image extensions
* `ext_video` (default `mp4 webm`) — allowed video extensions
* `ext_audio` (default `3ga aac aiff flac m4a mp3 ogg wav`) — allowed audio extensions
* `ns` (default `.`) — CLI namespace prefix
